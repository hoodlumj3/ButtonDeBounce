# ButtonDeBounce
Arduino code for button deBounce via inPin and / or ps3Nav SHADOW

Adds Extra button functionality to the SHADOW system 
like
Click
DoubleClick
Hold
LongHold

Button timings can be altered in code 
See .ino file for more info.
